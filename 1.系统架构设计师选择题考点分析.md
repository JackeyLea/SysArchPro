## 二、数据库系统（第4章）- 高频考点

### 2.1 数据库设计（中频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 数据库设计阶段 | 中频 | 需求分析、概念设计（E-R模型）、逻辑设计（关系模型）、物理设计 |
| E-R图集成冲突 | 中频 | 属性冲突、命名冲突、结构冲突 |
| 数据库转储 | 低频 | 静态/动态、全局/增量 |

### 2.2 数据库基本概念（高频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 三级模式结构 | 高频 | 外模式（用户视图）、概念模式（逻辑结构）、内模式（物理存储） |
| 两级映像 | 中频 | 外模式/概念模式映像、概念模式/内模式映像 |
| 数据独立性 | 中频 | 逻辑独立性、物理独立性 |

### 2.3 关系数据库（高频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 关系代数运算 | 高频 | 选择、投影、连接、并、交、差、笛卡尔积 |
| 自然连接 | 高频 | 同名属性等值连接、去除重复列 |
| 候选键判定 | 高频 | 属性闭包法、有向图法 |
| 函数依赖 | 高频 | 部分依赖、传递依赖 |
| Armstrong公理 | 中频 | 自反律、增广律、传递律、合并规则、分解规则、伪传递规则 |

### 2.4 数据库范式（高频）
| 范式 | 重要程度 | 说明 |
|------|----------|------|
| 1NF | 高频 | 属性原子性 |
| 2NF | 高频 | 消除非主属性对码的部分依赖 |
| 3NF | 高频 | 消除非主属性对码的传递依赖 |
| BCNF | 中频 | 消除主属性对码的部分和传递依赖 |
| 4NF | 中频 | 消除多值依赖 |

### 2.5 事务与并发控制（中频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| ACID特性 | 高频 | 原子性、一致性、隔离性、持久性 |
| 锁机制 | 中频 | 共享锁（S锁）、排他锁（X锁） |
| 两段锁协议 | 低频 | 加锁阶段、解锁阶段 |

### 2.6 特殊数据库系统（中频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 分布式数据库 | 中频 | 分片透明、复制透明、位置透明、逻辑透明 |
| 两阶段提交 | 中频 | 表决阶段、执行阶段 |
| 数据仓库 | 中频 | 面向主题、集成性、相对稳定性、反映历史变化 |
| 嵌入式数据库 | 低频 | 轻量级、实时处理、多线程并发 |

## 三、计算机组成原理（第5章）- 中频考点

### 3.1 基本概念（中频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 分级存储体系 | 中频 | 寄存器→Cache→主存→外存 |

| DMA | 中频 | 直接存储器访问，主存与外设之间 |

| 总线类型 | 低频 | 半双工、全双工 |

### 3.2 嵌入式系统（中频）
| 考点 | 重要程度 | 说明 |

| 嵌入式处理器类型 | 中频 | MPU、MCU、DSP、SoC |
| DSP特点 | 中频 | 哈佛结构、数字信号处理 |

| NPU | 中频 | 神经网络处理器 |
| 芯片等级 | 低频 | 民用级(0-70°C)、工业级(-40-85°C)、军用级(-55-125°C) |

### 3.3 流水线（中频）
| 考点 | 重要程度 | 说明 |

| 吞吐率 | 中频 | 单位时间完成指令数 |
| 加速比 | 中频 | 顺序执行时间 / 流水线执行时间 |

### 3.4 存储器（低频）

## 考点频率统计汇总

### 高频考点（每年必考）
1. 软件架构风格
2. ABSD方法与架构评估（SAAM、ATAM）
3. 4+1视图模型
4. 软件质量属性与架构策略
5. 数据库范式与关系代数
6. 设计模式
7. 面向对象分析与设计（UML）
8. 软件测试方法
9. 网络安全协议
10. 加密技术与安全机制

### 中频考点（经常考察）
1. DSSA
2. 新技术架构（微服务、云计算、物联网）
3. 数据库设计
4. 分布式数据库
5. 嵌入式系统
6. 流水线计算
7. 网络规划与设计
8. 项目管理（挣值管理）
9. 知识产权

### 低频考点（偶尔考察）
1. 特定领域架构细节
2. 存储器类型
3. 综合布线
4. 排队论
5. 专业英语
