# 系统架构设计师选择题考点分析报告

## 三、计算机组成原理（第5章）- 中频考点

### 3.1 基本概念（中频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 分级存储体系 | 中频 | 寄存器→Cache→主存→外存 |
| RISC与CISC | 中频 | 精简指令集 vs 复杂指令集 |
| DMA | 中频 | 直接存储器访问，主存与外设之间 |
| 通道 | 低频 | I/O处理机 |
| 总线类型 | 低频 | 半双工、全双工 |

### 3.2 嵌入式系统（中频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 嵌入式处理器类型 | 中频 | MPU、MCU、DSP、SoC |
| DSP特点 | 中频 | 哈佛结构、数字信号处理 |
| AI芯片 | 中频 | GPU、FPGA、ASIC |
| NPU | 中频 | 神经网络处理器 |
| 芯片等级 | 低频 | 民用级(0-70°C)、工业级(-40-85°C)、军用级(-55-125°C) |

### 3.3 流水线（中频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 流水线周期 | 中频 | 最长阶段时间 |
| 流水线执行时间 | 中频 | 第一条指令时间 + (n-1) × 流水线周期 |
| 吞吐率 | 中频 | 单位时间完成指令数 |
| 加速比 | 中频 | 顺序执行时间 / 流水线执行时间 |

### 3.4 存储器（低频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| ROM类型 | 低频 | ROM、PROM、EPROM、EEPROM |
| RAID | 低频 | RAID 0/1/5/10等 |

## 四、计算机网络（第6章）- 高频考点

### 4.1 网络规划设计（中频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 三层网络架构 | 中频 | 核心层、汇聚层、接入层 |
| 层次化设计原则 | 中频 | 从接入层开始设计 |
| 网络冗余设计 | 中频 | 避免单点失效 |
| SDN架构 | 中频 | 应用层、控制层、转发层 |

### 4.2 网络存储（低频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| RAID 5容量计算 | 低频 | (n-1) × 最小硬盘容量 |

### 4.3 综合布线（低频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 六个子系统 | 低频 | 工作区、水平、干线、管理、设备间、建筑群 |

### 4.4 网络协议（高频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| OSI七层模型 | 高频 | 物理层、数据链路层、网络层、传输层、会话层、表示层、应用层 |
| TCP/IP模型 | 高频 | 网络接口层、网络层、传输层、应用层 |
| 各层设备 | 高频 | 物理层（中继器、集线器）、数据链路层（交换机）、网络层（路由器） |
| 各层协议 | 高频 | TCP/UDP（传输层）、IP/ICMP（网络层）、HTTP/FTP/SMTP（应用层） |
| 子网划分 | 高频 | 子网掩码、主机地址计算 |
| IPv6 | 中频 | 128位地址、单播/组播/任播 |
| DNS解析 | 中频 | 递归查询、迭代查询 |
| DHCP | 中频 | 动态主机配置协议 |
| SSL/TLS | 中频 | 保密性、完整性、可靠性 |
| 电子邮件安全 | 中频 | S/MIME、PGP |
| 奈奎斯特定理 | 中频 | 采样频率 ≥ 2 × 信号最高频率 |

## 五、信息安全（第7章）- 高频考点

### 5.1 安全基础（高频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 安全属性 | 高频 | 机密性、完整性、可用性、可控性、不可否认性 |
| 安全攻击类型 | 中频 | 中断、篡改、伪造、重放 |
| 安全机制 | 中频 | 加密、数字签名、访问控制、数据完整性、认证交换 |

### 5.2 加密技术（高频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 对称加密 | 高频 | DES、3DES、AES、IDEA |
| 非对称加密 | 高频 | RSA、ECC、ElGamal |
| 哈希函数 | 中频 | MD5、SHA-1、SHA-256 |
| 数字签名 | 高频 | 签名+加密、不可否认性 |
| PKI/CA | 中频 | 公钥基础设施、数字证书 |

### 5.3 网络安全（高频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 防火墙 | 高频 | 包过滤、状态检测、应用代理 |
| IDS/IPS | 中频 | 入侵检测/防御系统 |
| VPN | 中频 | IPSec、SSL VPN |
| 安全协议 | 高频 | SSL/TLS、HTTPS、IPSec |

### 5.4 系统安全（中频）
| 考点 | 重要程度 | 说明 |
|------|----------|------|
| 访问控制模型 | 中频 | DAC、MAC、RBAC、ABAC |
| 安全审计 | 中频 | 日志记录、行为分析 |
| 恶意代码 | 中频 | 病毒、蠕虫、木马、勒索软件 |
